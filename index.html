☁️ Google Sheets 연동 설정
여러 명이 동시에 데이터를 입력하고 공유할 수 있어요.
설정 완료 후 상단의 ☁️ 저장 / ☁️ 불러오기 버튼으로 동기화하세요.
Apps Script 배포 후 받은 웹앱 URL을 입력하세요
📖 설정 방법 (1회만)
① Google Sheets 만들기
  → sheets.google.com 에서 새 스프레드시트 생성 → 팀원 모두에게 편집 권한 공유

② Apps Script 설정
  → 스프레드시트에서 확장 프로그램 → Apps Script 클릭
  → 아래 코드 복사 버튼의 코드를 붙여넣기
  → 코드 2번째 줄 SHEET_ID에 스프레드시트 ID 입력 (URL의 /d/와 /edit 사이 문자열)
  → 배포 → 새 배포 → 웹앱 선택 → 액세스: 모든 사용자 → 배포
  → 배포 URL을 위 입력란에 붙여넣기

③ 팀원 공유
  → HTML 파일을 팀원에게 공유 → 각자 URL·이름 설정
  → 입력 후 ☁️ 저장 → 다른 팀원은 ☁️ 불러오기로 최신 데이터 수신
☁️ 동기화 이력
설정 후 동기화하면 이력이 표시됩니다.
⚙️ 설정 페이지에서 Google Apps Script URL을 먼저 입력해주세요.
